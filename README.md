# RNA-seq

## Overview

This repository contains a reproducible RNA-seq analysis workflow. It is designed for processing raw sequencing data through standard quality control, alignment, and quantification steps.

The workflow is based on the BioWDL RNA-seq pipeline:  
https://github.com/biowdl/RNA-seq/tree/develop
